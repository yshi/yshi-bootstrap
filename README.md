# YSHI Bootstrap Theme
This is a bootstrap 4 theme.

## Development
You can run a dev server on 0.0.0.0:8080 thusly:

```Shell
npm install -g webpack rimraf webpack-dev-server
npm install
npm run serve 
```

This is public-facing.

To build the project:

```shell
npm run build
```

Boom, files in `dist/`.
